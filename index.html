⚠ Segurança da sua chave:
A chave é salva localmente no seu navegador (não vai pra servidor). Se você usar este painel em outro PC, precisa configurar de novo. Para revogar, vai em console.anthropic.com e exclui a chave.
💾 Backup dos dados
Como seus dados ficam salvos apenas no navegador deste dispositivo, é fundamental fazer backup regularmente. Suba o arquivo .json no Google Drive para ter segurança.
Baixa um arquivo com TODOS os seus dados (lançamentos, cobranças, configurações). Salve no Google Drive ou pen drive.
Selecione um arquivo .json exportado anteriormente. Os dados atuais serão SUBSTITUÍDOS.
💡 Dica: Crie uma pasta "Painel Líder" no Google Drive Desktop e salve seus backups lá. Assim eles sincronizam automaticamente com a nuvem.

📅 Recomendação: Faça backup pelo menos 1x por semana.
